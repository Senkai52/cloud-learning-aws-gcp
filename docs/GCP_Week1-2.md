# GCP Woche 1–2: Grundlagen

## 🎯 Ziele
- Compute Engine VM starten.
- Cloud Storage Bucket anlegen.

---

## ⚙️ Übungen
- VM (e2-micro) starten, Firewall für SSH/HTTP.  
- Bucket erstellen:
```bash
gsutil mb gs://mein-bucket-$RANDOM
gsutil cp hello.txt gs://mein-bucket-XYZ/
```
